# Integrations

## Supabase
Usado para persistência de workspaces, leads, integrações, fila e billing.

## WhatsApp Cloud API
Webhook base pronto em `/api/webhooks/whatsapp`.

## Meta Instagram/Facebook
As credenciais já têm espaço reservado no `.env.example`. A ingestão real deve ser ligada ao webhook da Meta.

## Stripe
A base de billing está pronta em `/api/billing/checkout`, atualmente em modo stub para você conectar com suas chaves.
