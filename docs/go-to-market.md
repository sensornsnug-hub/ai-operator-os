# Go-to-market playbook

## Best first market

Pick one painful and measurable outcome:
- lost leads
- abandoned carts
- silent prospects
- slow proposal follow-up

Do not launch as "AI for everyone".

## Offer stack

### Entry
Setup fee between R$ 1.500 and R$ 5.000

### Recurring
R$ 197 to R$ 597 for software access

### Premium managed
R$ 1.990+ for done-for-you optimization

## Acquisition channels

- outbound on WhatsApp and Instagram
- direct demo outreach to local businesses
- short videos with proof
- landing page with one sharp promise

## Demo script

1. Show silent leads.
2. Show the agent waking them back up.
3. Show recovered revenue.
4. Show how the knowledge base improves replies.
5. Close with setup + monthly.

## Main objections

### “I already use ChatGPT”
Reply: this is not a chat app, it is an operating system tied to lead flow, knowledge and actions.

### “I do not trust automation”
Reply: you can start with approval mode and human escalation only.

### “Will this replace my team?”
Reply: no, it removes manual follow-up and speeds up response quality.
