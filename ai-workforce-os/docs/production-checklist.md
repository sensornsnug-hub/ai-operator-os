# Production checklist

## Security
- enforce auth
- add route protection
- add rate limiting
- verify webhook signatures
- create RLS policies per workspace
- redact sensitive logs
- add audit trail

## Reliability
- create background job queue
- add retries for provider calls
- add dead letter handling
- add provider failover rules
- add status page

## Product
- onboarding wizard
- empty states
- template gallery
- copy refinement
- analytics events
- upgrade prompts
- trial expiration flows

## Sales readiness
- pricing page
- terms and privacy policy
- support inbox
- payment recovery
- white-label configuration
